#### [剑指 Offer 14- I. 剪绳子](https://leetcode-cn.com/problems/jian-sheng-zi-lcof/)

> - 执行用时：1 ms, 在所有 Java 提交中击败了37.19%的用户

```java
class Solution {
    public int cuttingRope(int n) {
        int [] dp = new int [n + 1];
        dp[1] = 1;
        dp[2] = 1;
        for(int i = 3; i <= n; i++){
            for(int j = 2; j < i ; j++){
                dp[i] = Math.max(dp[i],Math.max(j * dp[i - j], j * (i - j)));
            }
        }
        return dp[n];
    }
}
```
