#### [42. 接雨水](https://leetcode-cn.com/problems/trapping-rain-water/)

```java
    public int trap(int[] height) {
        if(height.length == 0){
            return 0;
        }
        int GaoDu = Arrays.stream(height).max().getAsInt();
        int sum = 0;
        for(int j = 1; j<=GaoDu; j++){
            boolean isStart = false;
            int tempSum = 0;
        for(int i = 0; i < height.length; i++){
            if(isStart && height[i] < j){
                tempSum++;
            }
            if(height[i] >= j){
                sum += tempSum;
                tempSum = 0;
                isStart = true;
            }
        }
    }
    return sum;
    }
```
